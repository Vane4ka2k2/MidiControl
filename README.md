# MidiControl — Управление Windows через Arturia Minilab 3 🎛

Легковесное и высокопроизводительное C++ приложение для Windows (MSVC), превращающее MIDI-клавиатуру **Arturia Minilab 3** в персональный аудиомикшер, пульт управления медиафайлами и микрофоном.

## 🌟 Основные возможности

- 🎛 **Аудиомикшер приложений (WASAPI Core Audio)** — Раздельное управление громкостью браузеров, мессенджеров, плееров и общей системной громкости.
- 🎙 **Mute микрофона** — Быстрое включение и выключение микрофона нажатием одной кнопки/пэда во всей системе.
- 🎹 **Поддержка пэдов и энкодеров** — Управление плеером (Play/Pause, Next, Prev), вызов сниппера скриншотов (`Win+Shift+S`) и режим «Приглушения» (Smart Ducking 15%).
- 🖥 **Всплывающий графический индикатор (OSD Overlay)** — Красивая полупрозрачная плашка над панелью задач с отображением текущего уровня громкости и статусов.
- ⚙️ **Системный трей и Автозапуск** — Работа в фоне возле часов, быстрое меню по правому клику и интеграция с автозагрузкой Windows.
- 📄 **Конфигурируемость (`config.json`)** — Настройка процессов приложений, номеров нот и задержки OSD без необходимости перекомпиляции.
- 🗺 **Карта элементов (`MappingWindow`)** — Встроенное графическое окно-шпаргалка с полной картой функций клавиатуры.

## 🎛 Карта элементов по умолчанию

### Фейдеры (Faders):
- **Fader 1** (CC `82` / `14`) — Общая мастер-громкость Windows
- **Fader 2** (CC `83` / `15`) — Громкость браузеров (`Chrome`, `Edge`, `Firefox`, `Opera`...)
- **Fader 3** (CC `85` / `30`) — Громкость голосовой связи (`Discord`, `Telegram`, `Teams`...)
- **Fader 4** (CC `17` / `31`) — Громкость музыки и плееров (`Spotify`, `YandexMusic`, `VLC`...)

### Пэды (Pads 1–8):
- **Pad 1** (Нота `36` / CC `102`) — **Mute/Unmute Микрофона**
- **Pad 2** (Нота `37` / CC `103`) — Mute общего звука
- **Pad 3** (Нота `38` / CC `104`) — Воспроизведение / Пауза
- **Pad 4** (Нота `39` / CC `105`) — Следующий трек
- **Pad 5** (Нота `40` / CC `106`) — Предыдущий трек
- **Pad 6** (Нота `41` / CC `107`) — Приглушение звука (Smart Ducking 15%)
- **Pad 7** (Нота `42` / CC `108`) — Скриншот области экрана (`Win+Shift+S`)
- **Pad 8** (Нота `43` / CC `109`) — Альтернативный Mute микрофона

### Энкодеры (Encoders):
- **Encoder 1** (CC `74`) — Громкость активного фокусного окна

## 🛠 Сборка и запуск

### Требования:
- Компилятор MSVC (Visual Studio 2019 / 2022 / 2026)
- CMake 3.16+

### Сборка через 1-клик скрипт:
Запустите `build.bat` в папке проекта:
```cmd
build.bat
```

Исполняемый файл появится по пути:
`build\Release\MidiControl.exe`
